# 桌游游玩记录！

!!! note "关于「评价」"
    这里的「评价」基于个人的体验！
    
    4 ~ 5 分是非常喜欢的，2.5 ~ 3.5 分是愿意再玩的，1 ~ 2 分是还不错但是不想再玩的，1 分以下是后悔玩过的QWQ 
    
    另外 `-` 是表示因为各种原因暂时无法或者不足以给出评价的
